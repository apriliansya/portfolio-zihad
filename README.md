# Portfolio Zihad Arkhan

Website portfolio pribadi berbasis HTML dan CSS untuk menampilkan profil singkat, tombol aksi, tautan sosial media, statistik, foto profil, dan video background.

## Fitur

- Tampilan hero untuk perkenalan diri
- Background video dengan efek glassmorphism pada konten utama
- Foto profil berbentuk lingkaran
- Tombol "Hire me"
- Ikon sosial media menggunakan Boxicons
- Animasi masuk elemen menggunakan AOS
- Layout responsive untuk layar desktop dan mobile

## Teknologi

- HTML5
- CSS3
- AOS Animation
- Boxicons

## Struktur Proyek

```text
portfolio_zihad/
├── images/
│   ├── galaxy.mp4
│   └── zihad.webp
├── index.html
├── style.css
└── README.md
```

## Cara Menjalankan

Proyek ini adalah website statis, jadi tidak membutuhkan instalasi package.

1. Clone repository:

```bash
git clone https://github.com/username/portfolio_zihad.git
```

2. Masuk ke folder proyek:

```bash
cd portfolio_zihad
```

3. Buka `index.html` di browser.

Untuk development yang lebih nyaman, gunakan ekstensi Live Server di VS Code.

## Deployment

Website ini bisa di-deploy ke layanan static hosting seperti:

- GitHub Pages
- Netlify
- Vercel

Untuk GitHub Pages, aktifkan melalui menu:

```text
Settings > Pages > Deploy from a branch > main > /root
```

## Bagian yang Bisa Diubah

- Nama dan teks profil di `index.html`
- Link sosial media pada elemen ikon
- Foto profil di `images/zihad.webp`
- Video background di `images/galaxy.mp4`
- Warna, ukuran, layout, dan responsive design di `style.css`

## Catatan

File `index.html` memakai CDN untuk AOS dan Boxicons. Pastikan koneksi internet aktif agar animasi dan ikon tampil dengan benar.
